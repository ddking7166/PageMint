# Introduction

PageMint is a server-side JSX framework built on Hono. It renders full HTML on the server, caches that HTML by route key, and keeps browser JavaScript optional and small.

It is designed for content-heavy pages, internal tools, documentation sites, media indexes, and classic server-rendered applications.

For app configuration and environment files, see [Configuration](./config.md).

For complete function signatures, parameters, return values, and types, see [API Reference](./api.md).
